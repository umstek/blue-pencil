Execute

blue-pencil script


About execute:

This executes a function inside the script or optionally an external executable file.


Code (function):

This contains inside a function or a condition and executes a function.

<execute function="#function1" content="$a1" key="$k1"/>

@function = name of the function to be executed.

@content = input to be passed to the function.

@key = second input to be passed to the function.


Code (application):

This contains inside a function or a condition and executes an application.

<execute appication="C:\Windows\explorer.exe" args="C:\"/>

@application = full path of the application to be executed.

@args = arguments (parameters/command) to be passed to the executable.
