Predefined Functions

blue-pencil script


Abput Predefined functions:

These functions are accessible to do certain tasks and they are default functions.


Code:

Used when defining a variable.

When defining (or setting) a variable using a function, all attributes after @content will be passed to the called function.


#rand

<define name="variableName" content="#rand" length="$default"/>

inputs:                length (how many random bytes to return)

output:        (cryptographically random data $length bytes)


#trim

<define name="newVariable" content="#trim" in="$variable"/>

inputs:                in (input data to trim)

output:        (leading zeros removed data)


#part

<define name="newVariable" content="#part" data="$variable" start="128" count="128"/>

inputs:                data (input data to get a part of it)

               start (starting position of the part)

               count (how many bytes to get)

output:        (part of the data)


#lengthof

<define name="newVariable" content="#lengthof" data="$variable"/>

inputs:                data (data to get the length of)

output:        (length of the given data in bytes)
