Condition

blue-pencil script


About condition:

Condition is the logical operator which executes different functions on different conditions.


Code:

Conditions contains inside a function.

One and only one statement must be written inside <greaterthan> or any other condition and it must be an <execute> command.

<condition select="$variable1">

       <greaterthan value="$variable2" break="true">

               <execute function="#function1" content="$a1"/>

       </greaterthan>

       <else>

               <execute function="#function2" content="$a2"/>

       </else>

</condition>

@select = selects the (first) variable to be compared with another(second).

Available conditions:        

                       <greaterthan> ; >

                       <lesserthan> ; <

                       <equal> ; =

                       <notgreaterthan> ; <=

                       <notlesserthan> ; >=

                       <notequal> ; <>

@value = second variable to be compared.

@break = whether not to execute further lines after executing first satisfied condition.

@function = name of a function to be executed.

@content = an input to the function, 'function1'
