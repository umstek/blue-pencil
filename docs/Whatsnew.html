WHAT'S NEW?

blue-pencil: Scriptable Shredding System


Everything except deleting the file!

blue-pencil is different from any other wiping or shredding software because of its scripting support.


What is it?

  • With blue-pencil  you can queue multiple files from different locations to be deleted.
  • The deletion procedure is not the common file deletion procedure but a procedure which uses cryptographically random data (if the algorithm supports) to overwrite the file multiple times before deletion.
  • (If the algorithm supports) blue-pencil renames the file before deletion.
  • You can create new algorithms (.bps files: blue-pencil script) if you have a general programming knowledge and/or you can customize default algorithms.
  • Supports wiping of free-space via creation on new files and applying the same procedure for file deletion.
