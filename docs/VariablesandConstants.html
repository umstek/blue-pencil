Variables and Constants

blue-pencil script


About  variables and constants:

These are memory locations to store data.


Code (definition):

Defining variable using size

<define name="x" length="4"/>

@name = name of the variable to be defined.

@length = length of the variable to be defined.


Code (definition):

Defining constant exclusively giving items.

<define name="x" content="4 255 32 65"/>

@name = name of the variable to be defined.

@content = decimal representation of the bytes of the constant.


Code (setting):

Setting constant exclusively giving items.

<set name="x" content="67 72 65 32"/>

@name = name of the variable to be set.

@content = decimal representation of the bytes of the constant.

if it is necessary to change the value of the variable, this is the code for that.


Code (disposing):

Clears variable and releases any resources used by the variable.

<dispose name="x"/>

@name = name of the variable to dispose.
