Event Functions

blue-pencil script


About  events:

Events are executed when the user does something during the process. (e.g.: Pressing pause button)


Code:

Events contain inside the <file> block or the <freespace> block.


<event_pause>

<pause/>

</event_pause>

Contains commands to be executed when user press the "Pause" button in main interface.


<event_resume>

<resume/>

</event_resume>

Contains commands to be executed when user Resumes the process.


<event_stop>

<terminate/>

</event_stop>

Contains commands to be executed when user Stops the process.


<event_finish>

<delete/>

</event_finish>

Contains commands to be executed when the overwriting process is completed.
