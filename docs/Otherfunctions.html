Other Functions

blue-pencil script


About other functions:

Other functions are functions you can define.


Code:

Functions contain inside the <file> block or the <freespace> block (events).

Functions can be accessed when defining a variable using #functionName


<fun1 in0="length" in1="accuracy">


<return content="$length*$accuracy"/>

</fun1>

@in0; @in1; ... = input names to the function.

<return> = returns an output if necessary.


<f in0="content" in1="timeout">

</display content="$content" timeout="$timeout">

</display content="$content" timeout="$timeout">

</f>

@in0; @in1; ... = input names to the function.

This function has no return. So it acts like a "Void" or "Sub".
