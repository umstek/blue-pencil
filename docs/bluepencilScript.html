What is blue-pencil script?

blue-pencil script is the language to create a .bps file which contains the algorithm to shred files. It's basically based on XML.


Is it a programming language?

You better call it a 'scripting' language rather than a programming language. Because normal programming languages (e.g.: VisualBasic.NET; delphi; Java; C) are compiled into a machine readable format before execution. But a script is executed just in time without a special compilation procedure (e.g.: vbscript; JavaScript; MS-DOS batch files).

So this is a script which tells the computer how to shred the file.


Starting to write a .bps file

blue-pencil script file is always surrounded with angle bracketed 'bluepencil' keyword.


<bluepencil fileversion="1.0" algorithm="CLR" algversion="1.0">

</bluepencil>

Let's consider XML attribute 'attribute' as @attribute and XML element 'element' as <element>... :-)


@fileversion = the version of the bps file.

@algorithm = the name of the algorithm.

@algversion = algorithm version.
